# Real Estate Data Analysis Project

## Project Overview
This project focuses on analyzing a real estate dataset to uncover pricing patterns, property characteristics, and location-based trends. The goal is to clean, structure, and analyze raw property data to generate actionable insights that can support data-driven decision-making in the real estate market.

## Objectives
- Clean and prepare raw real estate data for analysis
- Analyze property prices across different locations
- Examine the impact of property features such as bedrooms, bathrooms, and lot size on pricing
- Identify geographic trends in property values

## Tools & Technologies
- SQL
- PostgreSQL / MySQL (relational database concepts)
- Data Cleaning
- Exploratory Data Analysis (EDA)

## Project Structure
- `schema.sql` – Database schema and table creation
- `data_cleaning.sql` – Data cleaning and transformation queries
- `analysis.sql` – Analytical queries used to extract insights

## Key Analysis Questions
- Which property types are the most expensive?
- Which states or cities have the highest property prices?
- How does lot size influence property prices across regions?
- What is the average price based on number of bedrooms and bathrooms?
